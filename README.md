# Mini SOC / Threat Detection Lab

A beginner cybersecurity project focused on detecting suspicious activity from Linux authentication logs using Python.

## Features
- Parse Linux authentication logs
- Detect failed SSH login attempts
- Generate brute-force alerts by IP address

## Technologies
- Python
- Linux
- Git/GitHub

## Goal
Build hands-on experience with:
- Python scripting
- Log analysis
- Detection engineering
- Security automation
